# JOSHGRAPHIX_WORLD Build Phases

## Phase 1 - Foundation

- Design system, content model, static data structure, SEO baseline, and admin dashboard preview.
- Future database schema drafted for Prisma and PostgreSQL.

## Phase 2 - Public Website

- Landing page, navigation, footer, about, services, testimonials, FAQ, contact, WhatsApp integration, portfolio categories, album preview, project viewer, and sample pricing.

## Phase 3 - Next.js Migration

- Move the static UI into Next.js App Router, TypeScript, Tailwind CSS, and Framer Motion.
- Replace `data.js` with server-loaded Prisma data.

## Phase 4 - Admin Management

- Secure login, category management, album management, project upload/editing, Cloudinary or Supabase Storage, image optimization, thumbnails, and WebP support.

## Phase 5 - Contact and Messaging

- Email notifications, message inbox, search, read/archive/reply status, export, spam protection, and reCAPTCHA or Cloudflare Turnstile.

## Phase 6 - Analytics, SEO, and Security

- Google Analytics 4, internal event analytics, sitemap generation, structured data, rate limiting, CSRF protection, XSS hardening, image validation, backups, and production monitoring.

## Phase 7 - Production Readiness

- Accessibility audit, responsive QA, Lighthouse optimization, deployment to Vercel/Railway/Render/Neon, documentation, and handover.
